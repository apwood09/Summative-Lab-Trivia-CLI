# About The Project 

# Key aspects 
* Scoring System 
* Questions Timed
* Interactive Question Seletion

# Features

* 1 point per question
* Game Over Display with Final Score 
* Color-coded Terminal Output

# Getting Started

# Prerequisites
* JavaScript 
* Package Manager 

# Installation
# Run the follwing in your terminal

1. Fork repo: 
    1. Click fork 
    2. leave name or change 
    3. click create fork 
    
2. Clone the repo in your terminal: 
    1. Green code button => Copy HTTPS
    2. git clone 'HTTPS Link' 

3. Install dependencies 
    1. npm install 

# Usuage 

# Run the follwing in your terminal to start the game: 

1. node game.js 

# Game Controls: 
* Arrow Keys to select answers 
* Press Enter to submit
